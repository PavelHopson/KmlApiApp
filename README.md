# KML API Application

## Описание
Этот проект представляет собой веб-API, который обрабатывает KML-файлы с полями и их центрами, и предоставляет следующие возможности:
- Получение всех полей
- Расчёт площади поля
- Расчёт расстояния до центра поля
- Проверка принадлежности точки полю

## Технологии
- ASP.NET Core
- C#
- KML
- Swagger

## Установка и запуск
```bash
# Установите .NET SDK
# Клонируйте репозиторий
git clone https://github.com/PavelHopson/KmlApiApp.git 
cd KmlApiApp

# Восстановите зависимости
dotnet restore

# Запустите проект
dotnet run
